# Station V2 Web App

Isolated Next.js scaffold for V2.

## Safety
- No connection to V1 runtime.
- No production data access by default.
- Built only inside `Station_V2/`.

## Planned routes
- `/dashboard`
- `/customers`
- `/payments`
- `/billing/entry`
- `/billing/preview`
- `/billing/approvals`
- `/billing/print`
- `/reports/overview`
- `/reports/monthly-bills`
- `/reports/monthly-collections`
- `/reports/loss-analysis`
- `/reports/audit`
- `/settings`

## Local run
```bash
npm install
npm run dev
```
